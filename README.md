*Connection Diagram*

| Rainfall Sensor Pin | Wire Label on Board | Arduino UNO Pin | Description    |
| ------------------- | ------------------- | --------------- | -------------- |
| **VCC (+)**         | `+`                 | **5V**          | Power supply   |
| **GND (−)**         | `-`                 | **GND**         | Ground         |
| **SDA**             | `D/T`               | **A4 (SDA)**    | I²C data line  |
| **SCL**             | `C/R`               | **A5 (SCL)**    | I²C clock line |
